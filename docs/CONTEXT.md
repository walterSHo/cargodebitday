# CONTEXT

## Project overview
Статический single-page B2B-калькулятор Cargo Parts (HTML/CSS/JS) для:
1) середнього відтермінування,
2) мотивації / ЗП (місяць + квартал).

Основные точки публикации: `index.html` и `docs/index.html`.

## Current implemented logic
- Delay-калькулятор: борг / сума оборота / дні, валидация, copy, clear.
- Monthly:
  - бонус ТП,
  - CRM checkbox,
  - ступенчатый бонус плана,
  - снижение при скидке >= 27.5%,
  - бонус за відтермінування,
  - просрочка 3 диапазона в EUR + конвертация по курсу.
- Quarterly:
  - шины (минимальный план + over-plan),
  - 12 групп (включая PROFIT/JOKER),
  - average по 3 месяцам,
  - выполнено при average >= 95,
  - бонус 7/8/9/10+, +10000 при всех 12.
- Summary разделен на месяц / квартал / общий итог.

## Current UI state
- Строгий B2B стиль Cargo Parts.
- Monthly-grid уже выравнивался.
- Есть inline validation/messages, hints, progress по квартальным группам.
- Есть короткое и детальное копирование, print/PDF через `window.print()`.
- Есть дисклеймер.

## Important decisions
- CRM переведен на checkbox-логику (без отдельных полей звонков/встреч).
- Месячная и квартальная логика строго разделены.
- Квартал считается через средний % за 3 месяца.
- Пороги бонусов по группам уточнены: 7/8/9/10+ и все 12.
- Просрочка считается по 3 EUR-диапазонам.
- Улучшения делались итеративно, узкими задачами.

## Known caveats
- Автокурс EUR может быть недоступен (есть ручной fallback).
- Clipboard в ограниченных средах может падать (есть fallback через textarea/execCommand).
- `index.html` и `docs/index.html` нужно держать синхронными.

## Performance notes (2026-04-20)
- Пересчеты по input теперь идут через rAF-batching (меньше дергания при быстром вводе).
- Квартальный блок использует кэш DOM-узлов вместо повторных `querySelector` на каждый цикл.
- Summary-анимация без forced reflow для более плавного обновления.
